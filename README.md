# Mood Archive — Culture Diary MVP

포트폴리오용으로 제작한 개인 문화 기록 웹앱 MVP입니다.

## 실행 방법

1. 압축을 해제합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 기록 등록, 아카이브, 상세 보기, 리포트 기능을 확인합니다.

별도 서버나 설치 과정이 필요 없습니다.

## 포함 기능

- Dashboard: 전체 기록, 이번 달 기록, 대표 카테고리, 평균 평점 요약
- Archive: 기록 카드 목록, 검색, 카테고리 필터, 감정 태그 필터, 정렬
- New Record: 문화 기록 등록/수정
- Detail: 기록 상세 보기, 수정, 삭제
- Mood Report: 카테고리별 기록, 감정 태그 순위, 취향 문장
- localStorage 저장: 브라우저에 기록 저장
- 기본 비공개 컨셉 반영

## 포트폴리오 기획 포인트

- 노션/인스타그램과 다른 개인형 문화 아카이브 컨셉
- 모든 기록 기본 비공개
- 문화 경험을 카테고리, 날짜, 감정 태그 기준으로 재탐색
- 기록이 쌓이면 간단한 취향 리포트로 연결

## 실제 운영 확장 방향

- React 또는 Next.js 전환
- Supabase Auth / Database 연동
- Supabase Storage 이미지 분리 저장
- RLS로 사용자별 기록 보호
- 공개용 기록 테이블 별도 분리
- 연간 문화 리포트 / PDF 내보내기 확장
